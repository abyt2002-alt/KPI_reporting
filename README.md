# Report Maker

A beautiful React + TypeScript app for creating visual reports from CSV data and exporting to PowerPoint.

## Features

- 📤 Upload CSV files with auto-detection of column types
- 📊 Create line, bar, and pie charts
- 📝 Add text comments and images
- 🎨 Visual report builder with drag-and-drop reordering
- 📥 Export to professional PowerPoint presentations

## Setup

```bash
cd report-maker
npm install
npm run dev
```

## Tech Stack

- React + TypeScript (Vite)
- TailwindCSS
- Zustand (state management)
- Recharts (charts)
- PapaParse (CSV parsing)
- PPTXGenJS (PowerPoint export)
- html-to-image (chart capture)
- Lucide React (icons)

## Usage

1. **Upload** - Drag & drop a CSV file
2. **Charts** - Create charts from your data
3. **Report** - Arrange charts, text, and images
4. **Export** - Download as PowerPoint
